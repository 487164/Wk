BL-MCB-001 [MEDIA] [ESPUBLICO] [DHCP] Conexión de dispositivo no autorizado a la red 

Descripción

Se ha detectado que el servidor DHCP le ha asignado una dirección IP a un equipo que no tiene permitido conectarse a la red

  • Nombre: Conexión de dispositivo no autorizado a la red
  • Criticidad: Media
  • Tiempo de Atención: 90 minutos
  • Tecnologías utilizadas: MAC Address

MITRE

  • TA0001 - Initial Access
  • TA0007 - Discovery

What to do

Se deberá identificar que la dirección MAC del dispositivo que ha generado la alerta es  88-A4-C2-40-6C-F5 o F4-A4-75-BA-37-EF las cuales pertenecen al equipo Wictor

Notas de cierre

Puntos mínimos que debe de tener un cierre

  • Nombre del equipo
  • Dirección MAC
  • DIrección IP asignada
  • Resumen del análisis

Matriz de escalado

Aunque la matriz de escalado debería ser la misma para todos los casos de uso, puede haber casos que por falta de capacidades, por dependencias de otros servicio o incluso por que así se acuerde para un caso concreto, se debe definir la matriz de escalado.

En horario laboral: Atiende N1 y escala a N2

Fuera del horario Atiende N0 y escala a Guardia

Campos Necesarios

Los campos mínimos que deben seleccionarse para llevar a cabo la investigación:

  • Start Time

  • Destination Asset Name
