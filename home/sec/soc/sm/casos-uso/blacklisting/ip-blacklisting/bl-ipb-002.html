BL-IPB-002 [MEDIA] [ESPUBLICO] [FW] Nodos TOR: Tráfico saliente detectado hacia redes TOR 

Descripción

Detecta tráfico L2R desde activos de la organización contra direcciones IP que pertenecen a las redes TOR.

  • Nombre: Nodos TOR: Tráfico saliente detectado hacia redes TOR
  • Criticidad: Media
  • Tiempo de Atención: 90 minutos
  • Tecnologías utilizadas: Logs firewall

MITRE

  • TA0011 – Command and Control
  • T1573 - Encrypted Channel
    • T1573.002 - Encrypted Channel: Asymmetric Cryptography
  • T1090 - Proxy
    • T1090.003 - Proxy: Multi-hop Proxy

What to do

Se identifica la dirección IP contra la que se están conectando, y cual es el activo que está realizando la conexión. Se analiza con las diferentes herramientas OSINT (Virustotal, X-Force, e.t.c)

Se comprueba la persistencia de las conexiones y que es lo que las está lanzando desde el posible equipo infectado. También el estado de las conexiones, para comprobar si el firewall las ha estado denegando o han sido satisfactorias. 

 

En caso de que todas las conexiones hayan sido rechazadas por el firewall, se cierra como FP, siempre y cuando se determine las causas del beaconing.
