CC-SNC-003 [MEDIA] [ESPUBLICO] [FW] Comunicación desde una Botnet conocida

Descripción

Esta alerta se activa cuando se detecta una comunicación desde una IP potencialmente maliciosa, la cual pertenece a una botnet conocida. De manera que nos alerte de un posible Command & Control. 

  • Nombre: Comunicación desde una botnet conocida
  • Criticidad: Media
  • Tiempo de Atención: 90 minutos
  • Tecnologías utilizadas: Logs locales

MITRE

  • TA0011 – Command and Control

What to do

Una vez detectada la alerta hay que revisar la IP de origen para comprobar su categorización por diversas fuentes de inteligencia, y poder ahondar a quién pertenece. También deberá revisarse toda la comunicación que pueda estar viéndose desde esa dirección IP y si la Destination IP está entablando comunicación.

En caso de no encontrar nada sospechoso lo definiríamos como FP descartando un incidente de seguridad y proporcionando la nota de cierre correspondiente con la investigación realizada.

Notas de cierre

Puntos mínimos que debe de tener un cierre

  • Source IP
  • Destination IP
  • Actividad de la Source IP
  • Resumen del análisis
