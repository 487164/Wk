RE-PAS-003 [BAJA] [ESPUBLICO] [WEB] Minería de datos

Descripción

Este caso de uso identifica el uso de agentes relacionados con actividades de crawling que intentan sacar información de alguno de nuestros servicios web, mediante la técnica conocida como web scrapping.

  • Nombre: Mineria de datos
  • Criticidad: Baja
  • Tiempo de Atención: 120 minutos
  • Tecnologías utilizadas: Nginx HTTP Server, Apache HTTP Server, Microsoft IIS

MITRE

  • TA0043 – Reconnaissance

What to do

Esta regla no genera alerta, la unica accion que hay asignada por el momento es que las IPs que cumplan con la lógica desarrollada, sean introducidas en el RS dedicado a minería de datos, llamado ‘RS_Blacklist_IOCs_IP_DataMining’.

Por el momento, no se está haciendo nada con el RS más allá de almacenar las IPs, por lo que no se están bloqueando ni tomando ninguna acción sobre ellas. 

Se ha designado el #Ticket78864 para la creación de esta regla.
