RE-PAS-004 [BAJA] [ESPUBLICO] [WEB] Mineria de datos desde España o Portugal

Descripción

Este caso de uso identifica el uso de agentes relacionados con actividades de crawling que intentan sacar información de alguno de nuestros servicios web, mediante la técnica conocida como web scrapping. En este caso se centra sobre IP con geolocalización en España o Portugal. 

  • Nombre: Mineria de datos
  • Criticidad: Baja
  • Tiempo de Atención: 120 minutos
  • Tecnologías utilizadas: Nginx HTTP Server, Apache HTTP Server, Microsoft IIS

MITRE

  • TA0043 – Reconnaissance

What to do

Esta regla, al contrario que la RE-PAS-003, sí que genera alerta. 

El objetivo no es bloquear ni meter en un RS las IPs de origen español o portugues, las cuales hacen uso de agentes relacionados con actividades de:
crawlers / bad robots / suspicious spiders / junk web-scrapers / malicious spammersResource Development  

Dado que trabajamos tanto con entidades de España como Portugal, se quiere evitar FP donde se bloqueen dichas entidades por error. Es por eso que solo se buscará monitorizar y estar al tanto de cualquier actividad sospechosa dentro de nuestra geolocalización de trabajo. Para más información, se especifica en el ticket:

https://soporte.espublico.it/Ticket/Display.html?id=79153
