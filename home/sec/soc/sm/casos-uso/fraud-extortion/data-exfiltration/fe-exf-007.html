FE-EXF-007 [CRITICA] [ESPUBLICO] [SOAR] Posible exposicion de credenciales Pool Admin

Descripción

En este caso de uso detecta una posible filtración de credenciales de usuarios Pool Admin, siendo reportados por la plataforma de IntelX.

  • Nombre: Posible exposición de credenciales Pool Admin
  • Criticidad: Critica
  • Tiempo de Atención: 15 minutos
  • Tecnologías utilizadas: Logs del SOAR

MITRE

  • TA0010 - Exfiltration

What to do

Se encargará de revisar la notificación que llega al buzón del SOC, para investigar la posible filtración en IntelX.

También se podrá revisar en https://intel.security.espublico.com/reports/YYYYMMDD/intelx/

Si se trata de una exposición de credenciales, se deberá revisar el excel de usuarios reportados por si se ha reportado recientemente:

 https://docs.google.com/spreadsheets/d/1NkLmStHp7YMMx1GC103HKPu522-TZ6YV/edit#gid=1866393291

Se le mandará un correo electrónico a cumplimientonormativo@espublico.com ( RECORDAR No incluir contraseña o cookies )

Es importante en IntelX revisar la notificación y archivos extraídos, ya que en ocasiones puede ser más grave de lo que en un primer momento parece. 

En caso de encontrar algo de ese tipo se debería poner como información adicional.

Notas de cierre

Puntos mínimos que debe de tener un cierre

  • DNI
  • Poner si se ha revisado o no
  • Si era un falso positivo o no

Matriz de escalado

Debe reportarse a cumplimiento normativo, con CC al director de seguridad y equipo de Intel.

Reglas de caso de uso

APPLY FE-EXF-007 [CRITICA] [ESPUBLICO] [SOAR] Posible exposicion de credenciales Pool Admin on events which are detected by the LOCAL system

AND when the event QID is one of the following (1008250050) Pool Admin found

Campos Necesarios

Los campos mínimos que deben seleccionarse para llevar a cabo la investigación:

  • username (custom)
  • domain (custom)

Cierre por defecto

Nota de cierre que se debería de rellenar a la hora de cerrar una alerta:

Falso positivo:

  • Se ha detectado una posible exposición de credenciales la cual tras revisar se identifica que se trata de un falso positivo.

Positivo:

  • Se ha identificado una posible exposición de datos la cual tras revisar se ha observado que se trata de un positivo. Se reporta el/los usuario/s a cumplimiento normativo para su correspondiente deshabilitado y cambio de contraseña.
