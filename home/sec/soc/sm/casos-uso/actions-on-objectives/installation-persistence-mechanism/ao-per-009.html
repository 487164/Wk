AO-PER-009 [MEDIA] [ESPUBLICO] [WINDOWS] Modificación de tarea programada para ejecutarse con máximos privilegios

Descripción

Detecta cuando se modifica una tarea programada con permisos elevados para ejecutarse.

  • Nombre: Modificación de tarea programada para ejecutarse con máximos privilegios
  • Criticidad: Media
  • Tiempo de Atención: 90 minutos
  • Tecnologías utilizadas: Microsoft Windows Security Event Log

MITRE

  • TA0003 - Persistence
  • T1053 - Scheduled Task

 

What to do

Se deberá identificar qué tarea programada con privilegios elevados ha sido modificada y que modificación se ha realizado en ella, comparando la versión antigua con la nueva. Tras obtener esta información se tendrá que averiguar el motivo y el usuario que la ha modificado, para identificar que no se trata de ningún proceso con fines de persistencia que ha realizado la acción.
