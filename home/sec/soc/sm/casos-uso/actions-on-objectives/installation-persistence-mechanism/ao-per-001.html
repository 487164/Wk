AO-PER-001 [ALTA] [ESPUBLICO] [MULTI] Modificación de ficheros o directorios críticos

Descripción

Detecta la modificación de permisos de ficheros o directorios críticos

  • Nombre: Modificación de ficheros o directorios críticos
  • Criticidad: Alta
  • Tiempo de Atención: 30 minutos
  • Tecnologías utilizadas: Logs Locales

MITRE

  • TA0040 - Impact

 

What to do

Se deberá identificar el path completo del archivo/directorio al cual se le han modificado los permisos y la máquina en la que se encuentra dicho archivo/directorio. Una vez identificado, deberemos obtener la información del usuario que ha realizado dichas modificaciones, username, source IP y las correspondientes modificaciones realizadas. Posteriormente, el usuario deberá justificar dichas modificaciones para descartar el incidente de seguridad

Notas de cierre

Puntos mínimos que debe de tener un cierre:

  • Usuario que ha realizado las modificaciones
  • IP de origen de las acciones realizadas
  • Archivo/Directorio modificado
  • IP de la máquina en la que se encuentra el archivo/directorio
  • Acciones realizadas
  • Resumen del análisis

Matriz de escalado

Aunque la matriz de escalado debería ser la misma para todos los casos de uso, puede haber casos que por falta de capacidades, por dependencias de otros servicio o incluso por que así se acuerde para un caso concreto, se debe definir la matriz de escalado.
