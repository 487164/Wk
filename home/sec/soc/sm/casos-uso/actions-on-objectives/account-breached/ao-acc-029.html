AO-ACC-029 [MEDIA] [ESPUBLICO] [CMDB] Detectado login por parte del usuario glpi

Descripción

Este caso de uso identifica el inicio de sesión del usuario glpi en la CMBD, el cual es un super admin.

  • Nombre: Login Successful glpi
  • Criticidad: Media
  • Tiempo de Atención: 90 minutos
  • Tecnologías utilizadas: CMDB

MITRE

  • TA0001 - Initial Access

What to do

Una vez identificada el inicio de sesión en la CMDB por parte del usuario glpi, habrá que comprobar la IP de origen para comprobar desde qué activo se ha realizado el login. 

Nadie debería hacer uso del usuario glpi, ya que es un super admin, por lo que habría que consultar si hay una razón justificada para hacer uso del mismo.

Cada usuario tiene su propio usuario nominal en la cmdb, por lo que deben hacer uso de los mismos para tener controlada la actividad.

Notas de cierre

Puntos mínimos que debe de tener un cierre

  • Fecha / Hora del login de glpi
  • IP de origen
  • Resumen del análisis
