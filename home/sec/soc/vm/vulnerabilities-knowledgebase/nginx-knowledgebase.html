Contamos con una versión antigua de NGINX en la máquina Warrior, el cual no se puede actualizar porque no hay soporte para la distribución que tiene. En cambio, sí que se aplican los Fix Pack de RedHat.

HTTP/2 está deshabilitado en los NGINX, que son los servidores web utilizados en la organización. Por lo que cualquier vulnerabilidad relacionada con dicho protocolo quedará rechazada.

INFO:

Se puede utilizar el comando curl -I http://objetivo.com para obtener la versión de HTTP en caso de duda
