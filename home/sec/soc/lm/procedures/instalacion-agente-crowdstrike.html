Instalación agente Crowdstrike

 

Descargar agente

Para la instalación del agente de crowstrike, nos lo tendremos que descargar desde la url https://falcon.eu-1.crowdstrike.com/hosts/sensor-downloads dependiendo de la versión del SO donde vamos a instalarlo.

 

Instalación agente en endpoints linux

Una vez descargado, lo copiamos al servidor en el que se va a instalar y ejecutaremos lo siguiente:

sudo yum localinstall falcon-sensor-xxxxxxxxxxx.rpm -y
sudo /opt/CrowdStrike/falconctl -s --cid=BA2D3475137C4117BD1369DD3C28450D-BD
sudo systemctl start falcon-sensor
sudo systemctl enable falcon-sensor
sudo /opt/CrowdStrike/falconctl -s --tags="xxxxxxxxxxxxxxx"
