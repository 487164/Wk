Instalación agente Crowdstrike

 

Descargar agente

Para la instalación del agente de crowstrike, nos lo tendremos que descargar desde la url https://falcon.eu-1.crowdstrike.com/hosts/sensor-downloads dependiendo de la versión/arquitectura del SO donde vamos a instalarlo.

En futuras versiones el proceso de instalación y configuración se realizará de forma automática con los scripts de setup. 

 

Instalación agente en endpoints linux

Una vez descargado, lo copiamos al servidor en el que se va a instalar y ejecutaremos lo siguiente:

sudo yum localinstall falcon-sensor-xxxxxxxxxxx.rpm
sudo /opt/CrowdStrike/falconctl -s --cid=BA2D3475137C4117BD1369DD3C28450D-BD
sudo /opt/CrowdStrike/falconctl -s --tags="xxxxxxxxxxxxxxx"
sudo systemctl enable --now falcon-sensor

 

En los tags se deberán incluir los siguientes:

  • Server → en caso de ser un servidor
  • Parte de la empresa → IT, SEC, TECH, SA
  • Localización → Zaragoza, Nabiax, Interxion, Antequera, Granada, AWS, Azure, GCP
  • Aplicación → Gestiona, esFirma
