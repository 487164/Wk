Instalación manual de paquetes

 

En caso de que falle la instalación de algún paquete durante el proceso de actualización automática, y siempre y cuando no se solucione al forzar la obtención de actualizaciones, se deberá proceder a revisar la versión actual del paquete desde la consola de QRadar.

rpm -qa | grep -i <search-term>

O mediante yum

yum search <search-term>

 

Para ver información del paquete se puede utilizar el siguiente comando:

yum info <search-term>

 

Una vez obtenida la versión se debe de acceder a fix central y descargar el paquete rpm de la versión necesaria.

Acceder a fixcentral a través de este enlace

Buscar por los siguientes parámetros:

  • Software → IBM Security QRadar SIEM
  • Version → Seleccionar la versión mayor actual
  • Platform → Linux

 

Una vez descargada, se deberá copiar a la consola de qradar e instalar de la siguiente forma:

yum install /path_to_package/package.rpm

 

En caso de querer reinstalar un paquete fallido

yum reinstall package.rpm

 

Una vez resuelta la dependencia se deberá forzar de nuevo la obtención e instalación de actualizaciones para que complete la instalación.

Esta actualización se deberá de realizar a través de la sección de autoupdates en el panel de administración de QRadar.
