Instalar driver conector JDBC para MySQL

 

En caso de necesitar conectar por JDBC a una base de datos en MySQL se deberá instalar el conector en la colectora necesaria.

Esto se debe a que el servidor con rol de collector es el encargado de conectar a dicha base de datos para solicitar los mismos.

Para realizar la instalación del conector se deberán seguir los siguientes pasos:

 

1. Descargar la última versión del conector desde la url de mysql, para ello se deberá descargar para plataforma independiente y en format tar.gz

https://dev.mysql.com/downloads/connector/j/

 

2. Extraer el contenido del fichero tar.gz descargado y renombrar el fichero .jar siguiente

mysql-connector-j-x.x.xx.jar

Al siguiente formato

mysql-connector-java-x.x.xx..jar

 

3. A continuación acceder a las consolas de qradar y copiar el fichero en las siguientes rutas:

/opt/qradar/jars

/opt/ibm/si/services/ecs-ec-ingress/eventgnosis/lib/q1labs/

 

4. Reiniciar servicio tomcat 

sudo systemctl restart tomcat

 

5. Una vez reiniciado y revisando que los servicios y aplicaciones han levantado correctamente reiniciar el servicio de recolección de logs

sudo systemctl restart ecs-ec ecs-ec-ingress

 

6. Completados los pasos, acceder a la aplicación de log source manager y crear un log source con conexión por jdbc
