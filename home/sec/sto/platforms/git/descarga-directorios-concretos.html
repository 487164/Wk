Descarga de directorios concretos

 

En git es posible descargar sólo ciertos directorios para evitar disponer de un repositorio compelto en el servidor.

Para ello se explican a continuación los pasos necesarios a la hora de hacer el clonado del repositorio.

 

En primer lugar acceder al directorio donde se desea disponer del repositorio, crear el directorio y acceder al mismo

mkdir DIRNAME && cd DIRNAME

Iniciar git

git init

Añadir el track del repositorio

git remote add -f origin ssh://git@YOUR_HOST/YOUR_REPONAME/

Habilitar la característica de sparse checkout

git config core.sparseCheckout true

añadir los directorios deseados al fichero de configuración de sparse

echo 'YOUR_PATH' >> .git/info/sparse-checkout

Una vez definidos los directorios hacer un pull del repositorio 

git pull --set-upstream origin master

Una vez obtenido el código deseado se puede trabajar con normalidad para añadir ficheros

git add FILES

realizar un commit de los mismos

git commit -m "YOUR MESSAGE"

y subir los cambios realizados

git push --set-upstream origin master

Otra config que puede ser útil es evitar la subida de cambios en permisos o propietarios de ficheros y directorios, para ello lanzar el comando

git config core.fileMode false
