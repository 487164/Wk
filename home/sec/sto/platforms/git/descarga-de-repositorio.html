Descarga de repositorio

 

En caso de que se utilice linux o git bash en windows el proceso para configurar el acceso será el siguiente.

 

Configuración de la autenticación

En primer lugar se deberá crear un directorio con el nombre .ssh bajo la home del usuario (/home/user/.ssh en linux o C:\Users\user\.ssh en windows)

Una vez creado se deberá crear un fichero config dentro del mismo con el siguiente contenido

 Host NAME_WHAT_YOU_WANT
      HostName IP OR HOST
      User YOUR_USERNAME
      IdentityFile PATH/TO/PRIVATE/KEY

 

Descarga por ssh

Una vez creado acceder al directorio donde se quiera descargar el repositorio y lanzar el siguiente comando sustituyendo las siguientes partes:

  • YOUR_HOST → nombre del host que se ha puesto en el fichero .ssh (el valor en lugar de NAME_WHAT_YOUR_WANT)
  • YOUR_REPO →nombre del repositorio que se quiere descargar
git clone ssh://git@YOUR_HOST/YOUR_REPO

 

Descarga por http

En caso de que la descarga se realice por http se deberá solicitar un usuario de acceso, en caso contrario están almacenadas en teampass las credenciales del usuario nobody para la lectura de algunos de los repositorios por lo que no es necesario disponer de credenciales nominales.

Para descargar el repositorio ejecutar el siguiente comando

$ git -c http.sslVerify=false clone https://cm.security.espublico.com/git/REPO_NAME

Cuando se lance solicitará las credenciales como se ve en el siguiente ejemplo:
