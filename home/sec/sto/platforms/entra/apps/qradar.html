IBM QRadar Security

La app SOC_QRADAR_OFFICE365_INTEGRATION está en los tennant espublico.it y espublico.com

Se registra la nueva aplicación https://entra.microsoft.com/#view/Microsoft_AAD_RegisteredApps/ApplicationsListBlade

En Supported account types section se selecciona Accounts in this organizational directory only.

En Redirect URI (optional) seleccionar Web y escribir http://localhost

Registar la app y anotar el Application (client) ID, se necesitará para configurar el log source in QRadar.

Seleccionar Certificates & secrets > New client secret y un expiry period (2 años) y anotar el value.

En API permissions añadir:

Device Support Module (DSM)

  • Office 365 Management APIs > ActivityFeed.Read (Delegated & Application) 
  • Office 365 Management APIs > ActivityFeed.ReadDlp (Delegated & Application) 
  • Office 365 Management APIs > ServiceHealth.Read (Delegated & Application) 

Office 365 Message Trace REST API protocol for IBM Security QRadar

  • Office 365 Exchange Online > ReportingWebService.Read.All (Application)

Hacer click en Grant admin consent for [ORG]
