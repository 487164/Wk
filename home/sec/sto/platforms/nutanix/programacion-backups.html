Programacion de backups

 

Acceder a la sección de data protection en nutanix.

Desde la sección de Async DR añadir un nuevo Protection Domain con los siguientes datos:

  • Name → itx-sec-01_{INSTANCE_NAME}
  • Entities → seleccionar la o las máquinas correspondientes para el backup
  • Schedule → Crear un nuevo schedule con los siguientes datos por defecto:
    • Repeat Weekly → Sunday
    • Hour → especificar un horario de madrugada a partir de la 1h am
    • Create application consistent snapshots → False (si se activa la máquina se apagará para realizar el snapshot)
    • Retention policy → guardar 1 snapshot
