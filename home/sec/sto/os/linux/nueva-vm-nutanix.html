Nueva VM en Nutanix seguridad

 

Para la creación de una vm acceder a la sección de VM en nutanix y pinchar en Create VM

 

Creación de nueva VM

Crear la máquina con los siguientes valores:

  • Name  -> nombre deseado
  • CPU  -> especificar el número de vCPUs
  • Memory  -> especificar la memoria necesaria en GB
  • Disk  -> añadir un disco en base a la imagen de rocky existente
    • Operation -> "Clone From Image Service"
    • Bus Type -> "SCSI"
    • Image -> "Centos7x-template"
  • Network Adapters -> añadir dos interfaces
    • ITX-Security-SVCS
    • ITX-Security-MGMT

 

Guardar los cambios y arrancar el servidor.

Configurar la red de la máquina con nmtui.

Configurar el hostname del servidor

hostnamectl set-hostname [fqdn]

Actualizar el setup con el siguiente script

/opt/iubaris/setup/bin/git-get-master.sh

Actualizar el fichero options.local 

vi /opt/iubaris/setup/bin/options.local

Y realizar las siguientes modificaciones

  • Modificar el setup, activando el mismo y configurando el servidor apropiado
  • Establecer el grupo de seguridad a “security” si no lo está
  • Revisar que las ips/dns de los servicios de mgmt son correctos (zabbix, mgmt, cmdb….)

Completada la revisión ejecutar el setup

/opt/iubaris/setup/bin/setup.sh

 

Completada esta parte (si ya está disponible) ejecutar los scripts de identity

/opt/espublico/identity/bin/identity.sh

 

Management services

Añadir el nuevo servidor en los siguientes servicios

  • Zabbix
  • Bacula (si procede)
  • Configurar rsyslog

Completada esta parte verificar en los siguientes servicios que los datos llegan correctamente

  • CMDB
  • SIEM

 

Creación registros DNS

Creación de los registros DNS para las direcciones de SVCS y MGMT y las direcciones PTR
